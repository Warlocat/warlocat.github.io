Green's function methods (Part V)

The main reference is a note by Prof. Julien Toulouse. For Hartree-Fock wavefunction or other single determinant wavefunction, we have (from Wick's theorem) \[ i^2 G(1,2;1',2') = iG(1,1')iG(2,2') - i G(1,2') iG(2,1') \] These two terms are usually referred to as direct (Hartree) and exchange (Fock) term, respectively. Substitute \(G_2\) into the definition of self energy \begin{aligned} \Sigma(1,2) = \Sigma_H + \Sigma_x = -i\delta(1,2)\int G(3,3^+) v(1,3) d3+ iG(1,2^+)v(1,2) \end{aligned} Or we can write them as \[ \Sigma_H(1,2) = \delta(\tau)\delta(x_1-x_2)\int dx_3 n(x_3, x_3) v(r_1, r_3) = \delta(\tau)\delta(x_1-x_2) v_H(r_1) \] \[ \Sigma_x(1,2) = -\delta(\tau) n(x_1,x_2) v(r_1, r_2) = \delta(\tau) v_x^{HF}(x_1, x_2) \] One can see they correspond to the Coulomb and exchange interaction in Hartree-Fock theory. The quasiparticle equation is then just the Hartree-Fock equation \[ h(r_1)\psi_k(x_1) + v_H(r_1) \psi_k(x_1) + \int dx_3 v_x^{HF}(x_1, x_3) \psi_k(x_3) = \epsilon_k \psi_k(x_1) \] and the corresponding frequency-domain Green's function is \begin{aligned} G_{HF}(x_1,x_2,\omega) = \sum_a^{vir} \frac{\psi_a(x_1)\psi_a^*(x_2)}{\omega - \epsilon_a + i0^+} + \sum_i^{occ} \frac{\psi_i(x_1)\psi_i^*(x_2)}{\omega - \epsilon_i - i0^+} \end{aligned} It is, of course, the same as the Green's function in Szabo's book (Eqn. 7.29) obtained from the HF orbitals.

The linear response function of HF wavefunction is then \[ \chi_{HF}(1,2;1',2') = -i G_{HF}(1,2') G_{HF}(2,1') \] with the frequency-domain representation \[ \chi_{HF}(x_1,x_2;x_1',x_2';\omega) = \sum_{ai}\frac{\psi_i^*(x_1')\psi_a^*(x_2')\psi_a(x_1)\psi_i(x_2)}{\omega-(\epsilon_a-\epsilon_i) + i0^+} - \sum_{ia}\frac{\psi_a^*(x_1')\psi_i^*(x_2')\psi_i(x_1)\psi_a(x_2)}{\omega-(\epsilon_i-\epsilon_a) - i0^+} \] The HF BSE kernel is \[ \Xi_H(1,2;1',2') = i \frac{\delta \Sigma_H(1,1')}{\delta G(2',2)} = \delta(1,1')\delta(2,2')v(1,2) \] \[ \Xi_x(1,2;1',2') = i \frac{\delta \Sigma_x(1,1')}{\delta G(2',2)} = -\delta(1,2')\delta(1',2)v(1,2) \] In molecular orbital basis, we can calculate the matrix elements of the linear response function as \begin{aligned} \chi_{pqrs}(\omega) &= \int\int\int\int dx_1 dx_2 dx_1' dx_2' \psi_p(x_1')\psi_q^*(x_1)\psi_s(x_2')\psi_r^*(x_2) \chi_{HF}(x_1,x_2;x_1',x_2';\omega)\\ &= \sum_{ai}\frac{\delta_{pi}\delta_{qa}\delta_{ri}\delta_{sa}}{\omega-(\epsilon_a-\epsilon_i)} + \sum_{ia}\frac{\delta_{pa}\delta_{qi}\delta_{ra}\delta_{si}}{-\omega-(\epsilon_a-\epsilon_i)} \end{aligned} We can see it is a "diagonal" matrix if we combine the first two indices and the last two indices and the joint index must be \(ia\) or \(ai\). We thus have \[ \chi_{HF}^{-1}(\omega) = -\left[ \left( \begin{array}{cc}\Delta\epsilon & 0 \\ 0 & \Delta\epsilon \end{array} \right) - \left( \begin{array}{cc}\omega & 0 \\ 0 & -\omega \end{array} \right) \right] \] For the BSE kernel, we have \begin{aligned} \Xi_{pqrs} &= \int\int\int\int dx_1 dx_2 dx_1' dx_2' \psi_p(x_1')\psi_q^*(x_1)\psi_s(x_2')\psi_r^*(x_2) (\Xi_H + \Xi_x)\\ & = \langle qr|ps\rangle - \langle qr|sp\rangle = \langle qr||ps\rangle \end{aligned} We can now write down the correlated linear response function \[ \chi^{-1}(\omega) = \chi_{HF}^{-1}(\omega) - \Xi = -\left[ \left( \begin{array}{cc}A & B \\ B^* & A^*\end{array} \right) - \left( \begin{array}{cc}\omega & 0 \\ 0 & -\omega \end{array} \right) \right] \] where \[ A_{ia,jb} = \delta_{ij}\delta_{ab}(\epsilon_a-\epsilon_i) + \langle aj||ib\rangle \] \[ B_{ia,jb} = \langle ab||ij\rangle \] As we know, the poles of \(\chi\) are the excitation energies. They correspond to the \(\omega\)'s that gives zero eigenvalues for \(\chi^{-1}\) matrix. This leading to the eigenvalue equation \[ \left( \begin{array}{cc}A & B \\ B^* & A^*\end{array} \right) \left( \begin{array}{c}X \\ Y\end{array} \right) = \omega \left( \begin{array}{cc}I & 0 \\ 0 & -I\end{array} \right) \left( \begin{array}{c}X \\ Y\end{array} \right) \] Readers may recognize this is actually the TDHF eigenvalue equation.
